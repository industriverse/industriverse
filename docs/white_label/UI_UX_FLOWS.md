**Animations**:
1. **Particle Field**:
   - 80-100 particles
   - Slow, random movement
   - Connected by thin lines when close
   - Orange/purple/teal color mix
   - Opacity fades in/out (0.2-0.6)

2. **3D Platform Visual**:
   - Auto-rotate (2 RPM)
   - Interconnected nodes
   - Energy pulses along connections
   - Hover to stop rotation
   - Click nodes to see widget preview

3. **Text Entrance**:
   - Logo: Fade in + scale (0.8â†’1.0)
   - Headline: Slide up from bottom
   - Subheadline: Fade in with delay
   - Buttons: Slide up with stagger

**Interaction**:
- "Get Started" button:
  - Hover: Glow intensifies, lifts up 4px
  - Click: Smooth scroll to pricing section
  - Shimmer effect on hover (light sweep across)

- "View Live Demo" button:
  - Hover: Border glows orange
  - Click: Opens demo in modal/new tab

**Card Design**:
- **Background**: Glassmorphism (rgba(26, 31, 58, 0.7))
- **Border**: 1px solid rgba(255, 255, 255, 0.1)
- **Backdrop Filter**: blur(20px)
- **Padding**: 3rem
- **Border Radius**: 1.5rem
- **Shadow**: 0 20px 40px rgba(0, 0, 0, 0.3)

**Icon Treatment**:
```css
.value-icon {
  width: 80px;
  height: 80px;
  margin-bottom: 2rem;
  background: linear-gradient(135deg, #FF6B35, #9B59B6);
  border-radius: 1rem;
  display: flex;
  align-items: center;
  justify-content: center;
  box-shadow: 0 0 30px rgba(255, 107, 53, 0.5);
  animation: pulse-glow 2s ease-in-out infinite;
}

@keyframes pulse-glow {
  0%, 100% { box-shadow: 0 0 20px rgba(255, 107, 53, 0.3); }
  50% { box-shadow: 0 0 40px rgba(255, 107, 53, 0.6); }
}
```

**Scroll Animation**:
- Cards start with `opacity: 0, y: 40`
- As user scrolls into view, animate to `opacity: 1, y: 0`
- Stagger delay: 150ms between each card
- Use IntersectionObserver for trigger

## Accessibility

### WCAG 2.1 Level AA Compliance

**Color Contrast**:
- Text: Minimum 4.5:1 ratio
- Large text (18px+): Minimum 3:1 ratio
- Interactive elements: 3:1 ratio

**Keyboard Navigation**:
- All interactive elements focusable
- Visible focus indicators (3px outline)
- Logical tab order
- Skip navigation links

**Screen Readers**:
- Semantic HTML (nav, main, aside)
- ARIA labels for icons
- Alt text for images
- Live regions for updates

**Motion**:
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
